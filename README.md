# Haoran Liu — Academic Homepage

A portable, dependency-free academic website designed for GitHub Pages.

## Structure

- `index.html` — concise main home page
- `cv.html` — dedicated CV viewer
- `zh/index.html` — concise Chinese home page
- `assets/` — shared styles, scripts, and imagery

## Local preview

Serve the repository root with any static HTTP server. No build step is required.

## Publishing

The site uses relative URLs and is published through GitHub Pages from the repository's `main` branch.
