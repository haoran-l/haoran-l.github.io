# Haoran Liu — Academic Homepage

A portable, dependency-free academic website designed for GitHub Pages.

## Structure

- `index.html` — English home page
- `research.html` — research agenda and scholarly profiles
- `experience.html` — education, teaching, service, experience, and awards
- `about.html` — biography, contact, and interests beyond research
- `zh/index.html` — concise Chinese home page
- `assets/` — shared styles, scripts, and imagery

## Local preview

Serve the repository root with any static HTTP server. No build step is required.

## Publishing

The site uses relative URLs and can be published from a GitHub Pages user site or a project repository without changing the source structure. No GitHub remote is configured by this project.
